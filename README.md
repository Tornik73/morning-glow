## Installation

```bash
$ npm install
```

## Running the app

# IMPORTANT: 
# 1) on prod mode change USER_VERIFY_LINK and USER_RESET_PASSWORD_LINK to current production front-end link
# 2) package.json in "start:(prod or dev)" SET NODE_ENV=production&& (&& must be without spaces)!

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
